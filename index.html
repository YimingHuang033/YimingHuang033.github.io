Publications

MSV-PCT: Multi-Sparse-View Enhanced Transformer Framework for Salient Object Detection in Point Clouds
Zihao Wang, Yiming Huang, Gengyu Lyu, Yucheng Zhao, Ziyu Zhou, Bochen Xie, Zhen Yang, Yongjian Deng
AAAI '25

[Point Cloud, Multi-View] We propose a novel framework for Point Cloud Salient Object Detection (PCSOD) task, named MSV-PCT. Which leverage efficient multi-sparse-view branches to enhance the performance of transformer-based point cloud backbone on PCSOD task.
SDformer: Transformer with Spectral Filter and Dynamic Attention for Multivariate Time Series Long-term Forecasting
Ziyu Zhou, Gengyu Lyu, Yiming Huang, Zihao Wang, Ziyu Jia, Zhen Yang
IJCAI '24
paper / code

[Time-Series, Attention] We propose a novel Transformer architecture (named SDformer) for long-term time series forecasting. It is the first time to address the problem of smooth attention distribution when modeling time series data with a large number of variates.
Generic Attention-model Explainability by Weighted Relevance Accumulation

paper / sildes / video / code
